# RichPackage
Basic R functions: hello(), Rich_StdDev(), Rich_Variance()

Ensure you have internet access. 

Type in console
```
 library(devtools).
```
Type in console
```
 install.github("rtroise89/RichPackage")
```
Type in console
```
 library(RichPackage)
```
```
You now have access to package functions. Enjoy.
```
